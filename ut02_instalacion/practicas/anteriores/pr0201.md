# UT02: Instalación y configuración de un ERP

## PR0202: Preparación del entorno con Docker

### ¿Qué vamos a aprender?

En esta práctica vamos a ver si te han quedado claros los conceptos sobre Docker, para lo que tendrás que levantar un entorno con contenedores con las siguientes características:

- Odoo estará expuesto en el puerto 5000
- Únicamente estará mapeado en el equipo anfitrión el directorio con la base de datos de Postgres

### ¿Qué recursos me pueden ser útiles?

A continuación se indican algunos recursos que complementan los apuntes del módulo y que te pueden ser útiles para realizar esta práctica:

- [DockerHub](https://hub.docker.com/)
- [Markdown Cheatsheet](https://commonmark.org/help/)

### ¿Qué hay que hacer?

El objetivo será levantar lo siguientes contenedores:

- Uno con Odoo 16 
- Otro con Postgres16


### Opcional

Si tienes tiempo y quieres explorar más posibilidades de los contenedores, intenta añadir un tercer contenedor con pgAdmin para administrar la base de datos.

### ¿Qué hay que entregar?

Debes entregar:

- Debes documentar todos los pasos realizados en un documento en formato Markdown
